# Weather Icons

These local SVG assets are vendored from Meteocons by Bas Milius.

- Source: https://github.com/basmilius/meteocons
- Package/CDN: `@meteocons/svg-static` via jsDelivr
- License: MIT
- Style: static `fill` SVG icons

The app loads these files locally at runtime; it does not fetch weather icons from the network.
